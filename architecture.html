API Endpoints
GET/api/inventoryFilters: warehouse, category
GET/api/inventory/{id}Single SKU detail
GET/api/ordersFilters: warehouse, category, status, month
GET/api/orders/{id}Single order detail
GET/api/dashboard/summaryKPI aggregates — all filters
GET/api/demandDemand forecasts (no filters)
GET/api/backlogBacklog items + PO status flag
GET/api/spending/summaryTotal costs with % change
GET/api/spending/monthlyMonthly spend by cost type
GET/api/spending/categoriesSpend breakdown by category
GET/api/spending/transactionsTransaction log
GET/api/reports/quarterlyQuarterly performance stats
GET/api/reports/monthly-trendsMonth-over-month trends
POST/api/tasksCreate task
GET/api/tasksList all tasks
PATCH/api/tasks/{id}Toggle task completion
DEL/api/tasks/{id}Delete task
POST/api/purchase-ordersCreate purchase order
GET/api/purchase-orders/{backlogItemId}Get PO for backlog item
